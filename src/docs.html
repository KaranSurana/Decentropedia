Documentation For Content Area

Content Area supports HTML tags for editing

  • When you type h1 and press Tab it autocompletes to <h1> </h1>
    Then you can fill in the content between the tags.
  • Same autocomplete functionality is available for tags ranging from <h1> to <h6>.
  • Also, When you write img and then press Tab it autocompletes to <img src="">
    So, you can add image inside the content area as well part from the default image.
  • When you press enter it adds a <br> tag for line break.
  • More updates regarding autocomplete will be rolled out in future.
